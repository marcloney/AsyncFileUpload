AsyncFileUpload

AsyncFileUpload is a jQuery File Upload plug-in, utilising HTML5 drag-and-drop and ES5 File API for asynchronous file uploads.

Documentation is available from Github.

Download

  • AsyncFileUpload-0.0.2.tar.gz (Production)
  • AsyncFileUpload-0.0.2.tar.gz (Development)
You can also build from source: 
          $ git clone https://github.com/marcloney/AsyncFileUpload
          $ npm install && bower install
          $ grunt
